 Student Management System

About the Project
-The Student Management System is a simple project designed to manage student records efficiently. It allows users to store, update, and delete student information in an organized way.

 Features
 - Add new student details  
 - View all student records  
 - Update existing student information  
 - Delete student data  
 - Easy to use interface  

 Technologies Used
-  Programming Language: C++  
-  File Handling  

 How to Run
1. Clone this repository  
2. Open the project in any C++ compiler (like CodeBlocks / VS Code)  
3. Run the main file  

Future Improvements

-  Create graphical user interface (GUI)
